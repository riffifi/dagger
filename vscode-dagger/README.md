# Dagger Language Extension

Adds syntax highlighting and language support for Dagger source files (`.dag`).

## Features

- Syntax highlighting for Dagger keywords, directives, comments, strings, numbers, operators and identifiers.
- Language registration for `.dag` files.

## Installation

From the extension folder:

```bash
npm install
npm run package
```

This produces a `.vsix` file in the extension folder, which can be installed in VS Code.

## File Extension

The extension registers the `.dag` extension for Dagger source files.
